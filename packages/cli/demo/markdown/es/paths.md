# Formato Markdown

Markdown es un lenguaje de marcado ligero con sintaxis de formato de texto plano.

## ¿Qué es frontmatter?

Frontmatter es un bloque YAML en la parte superior de un archivo que contiene metadatos sobre el archivo.
