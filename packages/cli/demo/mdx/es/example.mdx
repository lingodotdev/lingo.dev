## Los bloques de código no están localizados


## El código en línea no está localizado

Este párrafo contiene `inline code that is not localized` y muestra cómo se preservan los fragmentos de código en línea.

Puedes usar `const variable = "not localized"` en tus ejemplos de código sin preocuparte por la localización.
