## Introduction

Show current authentication status and user email

## Usage

`lingo.dev auth [options]`

## Flags

### `--login`

```bash
lingo.dev auth --login
```

DEPRECATED: Shows deprecation warning and exits. Use \`lingo.dev login\` instead

### `--logout`

```bash
lingo.dev auth --logout
```

DEPRECATED: Shows deprecation warning and exits. Use \`lingo.dev logout\` instead
