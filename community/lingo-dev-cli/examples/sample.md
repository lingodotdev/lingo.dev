# Sample Documentation

This is a sample markdown file to demonstrate the translation capabilities.

## Features

- **Automatic Translation**: Translate files with a single command
- **Format Preservation**: Markdown formatting is preserved
- **Code Blocks**: Code blocks remain unchanged

Simply run the translate command with your desired languages.
