# Next.js AI Flashcard Language Learner

A fun, interactive flashcard app for language practice using **Lingo.dev JavaScript SDK** for real-time AI-powered translations.

## What the app does
- Shows English flashcards with meanings/examples.
- Translates the entire deck to selected language (Hindi, French, Spanish, etc.) on demand.
- Allows users to input custom English phrases for instant real-time translation (dynamic content demo).

## Prerequisites
- Node.js 18+
- Lingo.dev account & free API key (https://lingo.dev → dashboard)

## How to Run Locally
1. `npm install`
2. Create `.env.local` with `LINGODOTDEV_API_KEY=your_key_here`
3. `npm run dev`
4. Visit http://localhost:3000

## Highlighted Lingo.dev Features
- **JavaScript SDK** for runtime, per-request translation of objects/JSON/dynamic text.
- Contextual, natural AI translations (no manual dictionaries).
- Ideal for interactive/educational apps with user-generated or live content.
