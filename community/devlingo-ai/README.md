# DevLingo AI

**DevLingo AI** is a developer-focused demo tool that showcases **Lingo.dev localization workflows**.  
It converts **developer comments, error messages, and README lines** into **multi-language JSON files** ready for internationalization.

---

## ðŸ›  Features

### 1. Code Comment Translator
- Input multiple `// comments` from your code.
- Generates JSON keys automatically.
- Supports multiple languages (`en`, `hi`, `fr`).
- Copy or download the JSON output.

### 2. Error Translator
- Input developer error messages.
- Generates JSON keys and multi-language fields.
- Copy or download the JSON output.

### 3. README Translator
- Input README/documentation lines.
- Generates JSON keys and multi-language fields.
- Copy or download the JSON output.

### 4. Dashboard
- Switch easily between all 3 tools.
- Clean, responsive UI built with Tailwind CSS.
- Real-time output preview.
