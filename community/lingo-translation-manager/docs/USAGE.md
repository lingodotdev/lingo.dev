# Usage Guide

## Getting Started

1. **Install dependencies**:
```bash
   npm install
```

2. **Start development server**:
```bash
   npm run dev
```

3. **Open your browser** to `http://localhost:5173`

## Features Overview

### Translation Editor
- Search and filter translation keys
- Edit translations inline
- Add new translation keys
- Delete unused translations

### Live Preview
- See translations in a real UI
- Switch between languages instantly
- Preview how text flows in different languages

### AI Assistant
- Detects missing translations
- Provides AI-powered suggestions
- Bulk generation capabilities

### Export Settings
- Export to JSON, Properties, or YAML
- Download translations for use in your app

## Best Practices

1. Use namespaced keys (e.g., `button.submit`, `nav.home`)
2. Keep translations consistent across languages
3. Use the AI assistant to speed up translation work
4. Preview frequently to catch layout issues

## Integration with Lingo.dev

This tool demonstrates Lingo.dev integration patterns:
- Key-value translation structure
- Multi-language support
- Export formats compatible with Lingo.dev
