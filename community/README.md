
# Lingo Flashcards

An interactive, multilingual flashcard application built with React and Lingo.dev. This project allows users to learn new vocabulary by flipping cards to reveal AI-generated translations in real-time.



## Features

- 3D Flip Animation: Smooth CSS-based card interaction
- AI Translation: Real-time translation using the Lingo.dev SDK.
- Challenge Mode: Automatically picks a random target language to test your skills.
- Cross platform


## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express

**Localization:** Lingo.dev SDK

## Prerequisites

- Node.js
- Lingo.dev API Key
