# Community examples have moved 🚚

The community examples and demo apps that used to live here now have their own
repository:

👉 **https://github.com/lingodotdev/lingo.dev-community**

## Why?

These projects are standalone examples — they are **not** part of the published
Lingo.dev packages, the build, or CI. Moving them to a dedicated repo keeps the
main product repository focused on the core product and keeps its dependency
security surface clean.

The full history of every example (with original authorship) was preserved in
the new repository.

## Contributing an example

Open a pull request in
[lingo.dev-community](https://github.com/lingodotdev/lingo.dev-community)
instead of this repo.
