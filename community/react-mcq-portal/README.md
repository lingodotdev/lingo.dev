# React & Node.js MCQ Portal - Regional Localization (Hindi & Bengali)

A full-stack demo application showcasing how to localize an educational portal for regional Indian languages using the Lingo.dev CLI.

Used the **Lingo.dev CLI workflow** to manage translations:
1. **Extraction**: Defined UI strings in `frontend/src/locales/en.json`.
2. **Translation**: Used the CLI to generate `hi.json` and `bn.json` via Lingo CLI tool.
3. **Implementation**: Created a custom language provider in React to switch between the JSON buckets generated by the CLI.
