# 🌍 Lingo.dev Translation Manager

A powerful, visual translation management tool built with React that showcases Lingo.dev's capabilities.

## ✨ Features

- **Visual Translation Editor** - Edit translations with real-time search and filtering
- **Live Preview** - Instantly see how translations look in your app
- **AI-Powered Suggestions** - Automatic translation suggestions for missing keys
- **Missing Translation Detection** - Automatically identifies incomplete translations
- **Multi-Language Support** - Manage unlimited languages (EN, ES, FR, and more)
- **Export Options** - Export to JSON, Properties, or YAML formats
- **Beautiful Stats Dashboard** - Track completion percentage and coverage

## 🎯 Why This Project?

This demo showcases how Lingo.dev can be integrated into a visual workflow, making translation management accessible to non-technical team members while maintaining developer-friendly features.
