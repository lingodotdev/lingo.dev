Add community demo: Error Message & Validation Localization Engine

This PR adds a production-grade demo application to the `/community` directory that showcases how to automatically localize backend error and validation messages using Lingo.dev.

**Features:**
- Automatic language detection via query parameters and Accept-Language headers
- Real-time error message translation using Lingo.dev SDK
- Support for multiple languages (English, Spanish, French, German, Hindi, Japanese)
- Modern frontend UI with beautiful gradient design
- Mock mode for testing without API key

**Location:** `community/error-localization-engine/`
