Lingo.dev는 웹, 모바일 앱 및 마크다운 콘텐츠의 현지화 및 번역을 위해 LLM을 활용할 수 있도록 설계된 오픈소스 i18n 툴킷입니다.

Lingo.dev는 다음을 포함합니다:

1. **Lingo.dev CLI** - 앱과 마크다운 콘텐츠를 빠른 속도와 정확도로 번역하는 데 도움이 되는 CLI 도구입니다. [문서](https://lingo.dev/cli)
1. **Lingo.dev CI/CD** - 새로운 콘텐츠가 추가되는 즉시 자동으로 번역을 최신 상태로 유지하기 위해 구축된 GitHub, GitLab 및 Bitbucket용 CI/CD 통합입니다. [문서](https://lingo.dev/ci)
1. **Lingo.dev 컴파일러 🆕** - 기존 컴포넌트를 변경할 필요 없이 빌드 타임에 React 앱을 다국어로 만듭니다. [문서](https://lingo.dev/compiler)

모든 도구는 정확한 번역 및 현지화를 위해 LLM 모델을 활용하고 수동 작업을 제거하도록 설계되었습니다.

Lingo.dev 컴파일러는 기존 컴포넌트를 변경할 필요 없이 빌드 타임에 React 앱을 다국어로 만듭니다.

컴파일러를 실행하기만 하면 기존 LLM API 키를 사용하여 앱이 자동으로 여러 언어를 지원하게 됩니다.

데모를 시청한 후, 자세한 내용은 [문서](https://lingo.dev/compiler)를 확인하세요.
